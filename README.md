# Transparent Vibe 🌌

Кастомная тема для Spicetify с динамической настройкой прозрачности и поддержкой собственных фоновых изображений.

## Основные функции:
* **Динамическая прозрачность**: Настраивайте уровень прозрачности панелей (от 0 до 100%) через встроенное меню.
* **Эффект размытия (Blur)**: Умное размытие (Glassmorphism) для боковой и верхней панелей.
* [cite_start]**Кастомный фон**: Возможность менять фоновое изображение через URL или локальные файлы прямо в интерфейсе Spotify. [cite: 1, 3]

## Установка
1. Скачайте папку с темой в директорию `Themes` вашего Spicetify.
2. Примените тему:
   ```bash
   spicetify config current_theme transparent-vibe
   spicetify apply
